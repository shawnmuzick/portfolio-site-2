Goals For This Site

This site is an exploration of the HTML Custom Elements and WebComponent specs, and became an opportunity to replace the first iteration of my profile site.

View the old code here! View the new code here!

The experience has been interesting, as the spec feels to be "missing" certain features. The process woud have been more expeditious in a JS framework like React, but I feel the journey has been valuable. The resulting code is easily reusable, and has the performance benefits of small file sizes and built in browser functionality.

Canon

This is a multi-purpose web app developed for an employer. It includes analytics and forecasting with ChartJS used for the datavis. It also includes a graphic sales tag generator, a graphic catalog builder, an integration with the Reverb.com API to post products, and much more! The project is built using SQL server to integrate with the company's Point of Sale system. It also NodeJS, ExpressJS and TediousJS on the back end. The front end is built using HTML/CSS and plain JS. Note: many images contain portions redacted to respect the company's privacy
